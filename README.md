# TimeTrackerUi

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.0.8.

## Steps to run in docker
-- git clone 
-- cd TimeTrackerAllianz
-- npm install
-- ng build
-- cd timetracker
-- mvn clean install
-- docker-compose up

Then navigate to docker url with port 8888 in browser.
